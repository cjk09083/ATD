## ATD 프로젝트

## 목적
- 축산농장 모니터링 및 관리 
- 관리자 WEB, 키오스크, 전용 APP등 다양한 형태로 필요한 기능 지원

- 모니터링 서비스 앱

- 농장 상태, 그래프, 지도, 작업내역 관리
