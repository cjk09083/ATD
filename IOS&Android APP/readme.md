

# iOS/Android APP 

## 목적
- 축산농장 모니터링 및 작업내역 업로드 
- Flutter로 iOS, Android 하이브리앱 개발

## 기능 

### 농장 상태 모니터링 
- 로그인 후 농장별 서버와 연동하여 최신상태 출력 
- 사이드 메뉴에서 농장 선택 시 빠른 이동
